Demo App Setup Instruction
*********************************

Environment requirement Java1.8 and Maven
Build the project using maven command mvn clean install
Jar is created with name "oslobysykkel-client-1.0.0-SNAPSHOT.jar" under /target in project folder.

1)open command prompt and go to oslobysykkel-client folder
2) run the command "java -jar target\oslobysykkel-client-1.0.0-SNAPSHOT.jar"
3) Copy URL: http://localhost:8080/stations in any browser and enter
 it will show list stations with available locks and bysykkels
